# ✅ Java Quiz Template

## Quiz Metadata

| Field | Value |
| --- | --- |
| **Level** | Basic / Intermediate / Advanced / Expert |
| **Scope** | Per article / module / scenario |
| **Question count** | 3-5 / 10-15 / case-based |

## Template

```markdown
## 7. QUIZ

### Quick Check

1. Câu hỏi...
   - A. ...
   - B. ...

   - D. ...

2. Câu hỏi...
   - A. ...
   - B. ...
   - C. ...
   - D. ...

### Answer Key

1. **B** — Giải thích ngắn 1-3 câu.
2. **D** — Giải thích ngắn 1-3 câu.
```

## Quiz Quality Rules

- Không hỏi trivia vô nghĩa
- Ưu tiên câu hỏi buộc người đọc phân biệt concept gần nhau
- Với `advanced/expert`, thêm câu hỏi trade-off hoặc failure mode
